# Sendbird uikit gpt

## Requirements

- Node.js 18+
- Yarn 4+ (or corepack)

## Getting started

```bash
# Install dependencies
yarn install
```

## Development

```bash
# Start the app in development mode
yarn dev
```

- Built with [create-electron-vite](https://github.com/electron-vite/create-electron-vite)
